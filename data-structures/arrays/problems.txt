Basics & Array Manipulation

    Find the maximum and minimum element in an array.
    Reverse an array. -- done
    Find the “Kth” max and min element of an array. --done
    Move all the negative elements to one side of the array. --done
    Sort an array of 0s, 1s, and 2s without sorting algorithms (e.g., Dutch National Flag problem). -- done 
    Find the union and intersection of two sorted arrays. -- done
    Rotate an array by "K" elements. -- done

Intermediate Problems

    Find the largest contiguous subarray sum (Kadane’s Algorithm).
    Minimize the maximum difference between heights of towers in an array (Height Difference problem).
    Find duplicates in an array with optimal space complexity.
    Merge overlapping intervals.
    Find a subarray with a given sum.
    Count the number of inversions in an array.
    Rearrange an array in alternating positive and negative items, without changing the order.
    Find the "Kth" largest element in an unsorted array (use heaps or quickselect).
